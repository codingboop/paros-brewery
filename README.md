# Paros Brewery

A minimalist, Japanese-inspired website for Paros Brewery built with Astro.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Design Philosophy

This website embodies Japanese minimalism with:
- Clean typography and generous whitespace
- Subtle color palette inspired by natural materials
- Thoughtful use of negative space
- Serene and contemplative atmosphere
